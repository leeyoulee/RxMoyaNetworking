# RxMoyaNetworking

Function
1. 학교소식 Open API를 사용하여 데이터를 받아오는 부분을 RxMoya 네트워킹을 사용한 예제 앱
2. 원하는 학교 검색 후 등록하면 당일 급식 정보를 받아볼 수 있음
3. 당일 급식 정보가 없을 경우, 월별 급식 정보를 볼 수 있음
